## Overview

An interactive portfolio featuring a real-time **AI Personal Assistant** running on my physical hardware, showcasing the professional journey of **Lê Duy Thế (theld21)**.

## Tech Stack

### Frontend

- **Framework**: Next.js
- **Styling**: Tailwind CSS 4.0
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: Vercel

### Backend & AI

- **Server**: Node.js + Express.js
- **AI Model**: **Gemma-3-4b** (Local Inference)
- **Caching**: Node-cache (User session & history management)

### Infrastructure (The "Local Data Center" Setup)

- **Hardware**: **Mac Mini M4** (Self-hosted)
- **Connectivity**: **Cloudflared Tunnel** (Secure, portless routing)
- **Privacy First**: All AI calculations happen on my personal hardware—no data is sent to third-party cloud providers.
